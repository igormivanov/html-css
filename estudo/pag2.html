Resumo HTML

  • div = Divisões, servem para definir partes e que sejam identificadas mais facilmente.

  • b ou Strong = Transformar a palavra em negrito.

  • img = Adicionar uma imagem ( img src="##" alt="##" )

  • p = parágrafos ( tag de texto )

  • a = Gera um elemento clicável, que nos leva até uma outra página ( a + enter / target="_blank" serve para abrir em outra página ) Exemplo: Google

  • comentário = Mensagem para o dev, feita para deixar o code mais organizado ( crtl + / )

  • meta viewport e charaset = São respectivamente para dar suporte ao dispositivo mobile e para implementar caracteres especiais como ç.

  • ul = Unordered List --> Lista de itens SEM ordem rígida

  • ol = Ordered List --> Lista de itens COM ordem rígida

  • li = List Itens --> Itens da lista com ou sem ordem

  • br/ = Pular linha.

  • ! + enter = Já inicia o code com as principais tags montadas.

  • nav = Barra de navegação

  • dl = Definition List, lista de definições

  • SEO = Não é uma tag, apenas um conjunto de estratégias com o objetivo de potencializar e melhorar o posicionamento de um site nas páginas de resultados naturais (Search Engine Optimization)
